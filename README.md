## 🛠️ Troubleshooting

### The app does not open

- Make sure you extracted the zip file first
- Check that the download finished
- Try running the app from the extracted folder
- Restart Windows and try again

### The video will not load

- Try another file format
- Move the file to a simple folder like `C:\Videos`
- Remove special characters from the file name
- Check that the file is not open in another app

### Processing is slow

- Close other apps
- Use a shorter video
- Make sure your PC has enough free RAM
- Save files on a local drive, not a slow external drive

### Audio text is poor

- Use a clearer audio track
- Reduce background noise
- Test with a shorter clip first
- Make sure the speaker is close to the mic
